# lenslog-site

Public website for [LensLog](https://lenslog.codeshrew.com) — a private,
on-device contact lens tracker for iPhone.

Static HTML, published via GitHub Pages at `lenslog.codeshrew.com`.

- `index.html` — splash page
- `privacy/index.html` — privacy policy (source of truth:
  `docs/legal/privacy-policy.md` in the app repo; regenerate the body with
  pandoc when the policy changes)
